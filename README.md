# VMS-NC Google Sheets Integration

Локальное окружение для разработки скриптов Google Apps Script платформы VMS-NC.

## Требования

- Node.js >= 18.x
- npm >= 9.x

## Установка и настройка

### 1. Включите доступ к Google Apps Script API в настройках вашего Google-аккаунта:
   https://script.google.com/home/usersettings

### 2. Установите глобально утилиту Clasp (если она не установлена в системе):
   ```bash
   npm install -g @google/clasp
   ```

### 3. Авторизуйтесь в вашем Google-аккаунте через консоль:
   ```bash
   clasp login
   ```

### 4. Склонируйте репозиторий и установите локальные зависимости (типы автодополнения для IDE):
   ```bash
   npm install
   ```

### 5. Настройте файл конфигурации связи с таблицей. В корне проекта должен находиться файл `.clasp.json` со следующим содержимым:
   ```json
   {
     "scriptId": "ВАШ_SCRIPT_ID",
     "rootDir": "./"
   }
   ```
   *Script ID находится в веб-интерфейсе Apps Script в разделе «Настройки проекта» (Project Settings).*

## Процесс разработки

- **Скачать код из облака Google в локальную папку:**
  ```bash
  clasp pull
  ```

- **Отправить локальные изменения из IDE в облако:**
  ```bash
  clasp push
  ```

- **Запустить автоматическое отслеживание изменений и пуш при сохранении файлов:**
  ```bash
  clasp push -w
  ```
